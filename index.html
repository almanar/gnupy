gnu in python

This is Python implementation of some GNU utilities. The intention was to make them available on windows platform with minimal dependencies and installation procedure. The only thing they depends on is Python 2.7 interpreter. To use them on windows just unzip at some place and add the folder to the global PATH environment variable.

Do it by right clicking on My Computer, selecting Advanced System Settings and then press on Environment Variables.

Following utilities are available:

  • gzip, gunzip, zcat
  • zip, unzip
  • tar
  • grep
  • cat
  • curl
  • md5sum
  • dos2unix, unix2dos
  • More will follow ...
